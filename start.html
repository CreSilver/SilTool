Tohle je úvodní stránka SilverTool Prototype.

Tento nástroj slouží jako uložení dokumentace a práci s jednodychými nástroji.

Mezi dokumentaci matří dokumentace pro kompilaci, pro Assembler či C/C++.

Tahle aplikace byla dělána pomocí frameworku QT. Samotné prohlížení dokumentů se provádí pomocí QWebEngine, který funguje na chromiu, takže aplikace je schopna pracovat se soubory: HTML, CSS i JS.
