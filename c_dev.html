C/C++ dokumentation


Základní struktury

Toto je záhladní struktura veškerých C/C++ souborů.

Struktura programu:

 main.c; main.cc; main.cpp

// Knihovny
#include <stdio.h>
#include <stdlib.h>


// Hlavní funkce
main(){

  return 0;
}

Struktura tříd:

 main.cc; main.cpp

// Třída
class MyClass{
  private:
    int variables;


  public:
    NazevTridy(){ // Konstruktor
      // Nastavení výchozích hodnot
    }

    virtual ~NazevTridy(){ // Destruktor
      // Tady uklidíš paměť, zavřeš soubory atd.
    }

    void naseFunkce(){} // Funkce třídy
};// Uzavření třídy


Knihovny

Mezi mejzákladnější knihovny patří:

  • stdio.h - pro vstup a výstup
  • stdlib.h - pro práci s pamětí a konverzi
  • string.h - pro práci s řetězci
  • math.h - pro matematické funkce

Specificky pouze C++:

  • iostream.h - pro vstup a výstup (std::cin, std::cout)
  • Pro lepší práci s std je lepší vypsat příkaz jako samotný using namespace std;. Nefunguje bez iostream.h!
  • memory.h - chytré ukazatele (smart pointers), co za tebe mažou paměť
  • vector.h - pro dynamické pole (std::vector)

Mezi další dobré knihovny patří:

  • time.h - pro práci s časem
  • locale.h - pro práci s lokalizací / jazyky
  • stdbool.h - pro práci s logickými true, false
  • ctype.h - pro práci s typy znaků (malé písmeno, velké písmeno, atd.)
  • limits.h - definuje limity pro celočíselné typy (INT_MAX, atd.)
  • float.h - limity pro desetinná čísla
  • assert.h - pro diagnostiku a ladění (makro assert)

Specificky pouze C++:

  • pthread.h - vícevláknové programování (multithreading)
  • sys/socket.h - síťová komunikace (sockety)
  • signal.h - zpracování systémových signálů (např. Ctrl+C)
  • algorithm.h - pro algoritmy (std::sort, atd.)
