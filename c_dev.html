Základní struktury

Toto je záhladní struktura veškerých C/C++ souborů.

Struktura programu:

 main.c; main.cc; main.cpp

// Knihovny
#include <iostream.h> // Pro C++
#include <stdio.h> // Pro C
#include <stdlib.h>


// Hlavní funkce
main(){

  return 0;
}

Struktura tříd:

 main.cc; main.cpp

// Třída
class MyClass{
  private:
    int variables;


  public:
    NazevTridy(){ // Konstruktor
      // Nastavení výchozích hodnot
    }

    virtual ~NazevTridy(){ // Destruktor
      // Tady uklidíš paměť, zavřeš soubory atd.
    }

    void naseFunkce(){} // Funkce třídy
};// Uzavření třídy


Knihovny

Mezi mejzákladnější knihovny patří:

  • stdio.h - pro vstup a výstup
  • stdlib.h - pro práci s pamětí a konverzi
  • stdint.h - pro práci s Unit datovými typy
  • string.h - pro práci s řetězci
  • math.h - pro matematické funkce

Specificky pouze C++:

  • iostream.h - pro vstup a výstup (std::cin, std::cout). Používá se místo stdio.h i když je použitelná.
  • Pro lepší práci s std je lepší vypsat příkaz jako samotný using namespace std;. Nefunguje bez iostream.h!
  • memory.h - chytré ukazatele (smart pointers), co za tebe mažou paměť
  • vector.h - pro dynamické pole (std::vector)

Mezi další dobré knihovny patří:

  • time.h - pro práci s časem
  • locale.h - pro práci s lokalizací / jazyky
  • stdbool.h - pro práci s logickými true, false
  • ctype.h - pro práci s typy znaků (malé písmeno, velké písmeno, atd.)
  • limits.h - definuje limity pro celočíselné typy (INT_MAX, atd.)
  • float.h - limity pro desetinná čísla
  • assert.h - pro diagnostiku a ladění (makro assert)

Specificky pouze C++:

  • pthread.h - vícevláknové programování (multithreading)
  • sys/socket.h - síťová komunikace (sockety)
  • signal.h - zpracování systémových signálů (např. Ctrl+C)
  • algorithm.h - pro algoritmy (std::sort, atd.)

Speciální syntaxe C++

1) Scope Resolution Operator je označová jako :: a používá se pokud existuje více věcí se stejným názvem. Primárné se používá u 3 hlavních situací:

  • Přístup k namaspace std - std::cout - jdi do std a najdi tam cout
  • Definice metody třídy v .h souboru. Zdělujěš tím ke které třídě patří:
     main.cc; main.cpp

    // V hlavičce: class MojeTrida { void pozdrav(); };
    void MojeTrida::pozdrav(){}
  • Přístup ke globální proměnné - pokud máš v kodu globální a lokální proměnou se stejným názvem tak funkce vždy prioritizují lokální funkce, ALE pokud dáš před proměnou :: tak se to aoutomaticky odkáže na globální proměnou a lokální proměnná se nevyužije.

Platí jak pro C tak pro C++, ale primární využití je v C++

2) Komunikace s třídou / strukturou pomocí . a ->

  • -> se poyžívá jako ukazatel neboli pointer, když se něco hledá a provádí se to jinde
  • . se používá pokud je objekt tady a přítomen

Assembler dokumentace zde

Deklarace funkcí, které se budou dělat externě v ASM:

 main.c; main.cc; main.cpp

void MyFunc();

 main.c; main.cc; main.cpp

extern void MyFunc();

Deklarace proměnných:

Veškeré proměnné musí být deklarovány jako globální jinak k nim nebude mít Assembler přístup. Taktéž se neprovádí standartní deklarace obyčejně podle int, char .atd, ale obvykle pomocí Uint typů. To ale neznamená, že se pomocí nich jako je třeba char nelze deklarovat.

uint8_t
